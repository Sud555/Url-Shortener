# URL Shortener (Java - Spring Boot, JDBC)

## Features

- Anonymous URL shortening
- User registration/login
- Custom short URLs for logged-in users

## Tech Stack

- Java 17
- Spring Boot
- JDBC (No ORM)
- MySQL
- HTML/CSS frontend
- GitHub Actions for CI

## Build & Run

```bash
mvn clean install
mvn spring-boot:run
```

## CI/CD

GitHub Actions runs unit tests on PR and merge.
